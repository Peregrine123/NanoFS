## 关键Bug修复

### Bug #1: 根目录direct[0]使用逻辑块号
**问题**: mkfs.c创建根目录时，direct[0]存储的是逻辑块号0，而inode_read需要物理块号。

**修复**:
```c
// 修复前
root.direct[0] = 0;

// 修复后
root.direct[0] = sb->data_start;  // 物理块号
```

**影响**: 修复后readdir可以正常读取根目录的`.`和`..`目录项。
